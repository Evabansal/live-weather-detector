# live-weather-detector

A modern, responsive **Live Weather Detector Web App** built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the **OpenWeather API**.  
This project demonstrates API integration, asynchronous JavaScript, and clean UI design.



##  Features

- Search weather by **city name**
-  Displays **temperature (Â°C)**
-  Shows **weather condition**
-  Humidity percentage
-  Wind speed (m/s)
-  Real-time API data fetching
-  Responsive & clean UI

##  Tech Stack

- Frontend: HTML5, CSS3
 Logic: JavaScript (ES6+)
- API: OpenWeather API
- Tools: VS Code, Git, GitHub

index.html

## How It Works

1. User enters a city name
2. JavaScript sends a request to OpenWeather API
3. API responds with live weather data
4. UI updates dynamically without page reload
